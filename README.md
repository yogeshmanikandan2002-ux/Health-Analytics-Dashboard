#  Health Analytics Dashboard

## Overview

The Health Analytics Dashboard is a personal health monitoring and analytics solution developed using **Microsoft Power BI** and **Power Apps**. The project integrates health and activity data from wearable devices such as Fitbit and Android Smartwatches to provide meaningful insights into wellness, fitness, and recovery progress.

This project was inspired by a personal post-surgery recovery journey and demonstrates how data analytics can support informed health decisions through interactive visualizations and performance tracking.

---

## Features

###  Health Monitoring Dashboard

* Track key health metrics in a centralized dashboard.
* Monitor daily wellness trends and recovery progress.
* Interactive visualizations for easy analysis.

###  Activity Tracking

* Daily steps monitoring.
* Activity and fitness goal tracking.
* Progress visualization over time.

###  Recovery Analytics

* Analyze recovery performance using historical data.
* Identify trends and improvements through visual dashboards.
* Generate actionable insights from wearable device data.

###  Power Apps Integration

* User-friendly interface for accessing health information.
* Seamless integration with Microsoft Power Platform.
